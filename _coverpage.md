# 🍴 FockNote

> Own your fockin' notes.

A free, installable, **git-backed** personal notebook. Fork your own Notion — your notes
are Markdown files in **your** repo. No server, no subscription, no lock-in.

- 📝 Edit anywhere (mobile-friendly PWA)
- 🔒 Private by default — your notes, your private repo
- 💸 $0 forever on GitHub Pages + GitHub
